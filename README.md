# 随机点名器
### 一个简单易用的随机点名器。
**注意：本程序需要在下载后手动在根目录创建一个文件（默认文件名为names.txt），在这里放入要随机的名单，一行一个名字。（以后说不定会支持去空行和注释的撒(ﾉ･ω･)ﾉﾞ）**

![主界面截图](./截图/主界面.PNG)
*主界面截图*

点击“随机点名”按钮即可开始随机点名，需要的话可以在设置菜单中设置字体、动画开关等哦(○’ω’○)
